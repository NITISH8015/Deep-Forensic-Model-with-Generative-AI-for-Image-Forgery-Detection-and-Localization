About ForgeVision

ForgeVision is an intelligent Image Forgery Detection System designed to identify and analyze digital image manipulations with high accuracy. It helps users verify image authenticity by comparing an original image with a suspected forged version and visually highlighting tampered regions. The system combines Structural Similarity Index (SSIM) analysis, pixel-level difference mapping, heatmap visualization, and YOLO-based face localization to detect even subtle modifications such as face retouching, object insertion, background alteration, or image enhancement. ForgeVision not only detects forgery but also explains the results in a user-friendly manner, providing clear metrics like the number of edited regions, percentage of modified pixels, SSIM score, and a confidence level (Low / Medium / High Forgery). A downloadable forensic PDF report is generated automatically for documentation and verification purposes. This platform is suitable for digital forensics, media verification, academic research, journalism, and legal evidence validation, offering a reliable and transparent solution for image authenticity analysis.

  • ✔ SSIM-based structural analysis
  • ✔ Pixel-level manipulation detection
  • ✔ Heatmap visualization of edits
  • ✔ YOLO face-only localization
  • ✔ Region-wise forgery explanation
